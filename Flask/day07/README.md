## 三 前后端分离（AXF）
## 例如， 一个页面渲染后是100K，网速10KB/S
# 前后端不分离: 整个页面的内容获取完才能显示 [10s]
# 前后端分离:
#    - 基本骨架(10KB) [1s]
#    - 轮播图数据(1KB) [0.1s]    ==》 DOM操作   ==》   加载图片
#    - 导航数据(1KB) [0.1s]
- AXF(轮播图) [后台]
```
- 建模(表)
- 导入数据
- 逻辑处理
- 返回JSON
```

- AXF(轮播图) [前端]
```
- 项目架构
- 第三方引入
- 页面结构设计
- 轮播图HMTL&CSS&JS
- ajax获取数据
- DOM操作(根据所以写的基本结构进行调整)
```
